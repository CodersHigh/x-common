아래와 같은 ASCII 도형에서, 사각형의 수를 세는 프로그램을 구현해주세요.

```
   +--+
  ++  |
+-++--+
|  |  |
+--+--+
```

위의 도형은 6개의 사각형으 포함하고 있습니다:

```


+-----+
|     |
+-----+
```

```
   +--+
   |  |
   |  |
   |  |
   +--+
```

```
   +--+
   |  |
   +--+


```

```
       
       
   +--+
   |  |
   +--+
```

```
       
       
+--+
|  |
+--+
```

```
       
  ++   
  ++   
       
       
```

입력되는 도형이 항상 적절한 사각형이라고 가정할 수 있습니다.(즉, 모든 선의 길이는 첫번째 선의 길이와 동일합니다.)
